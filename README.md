## 四、项目结构
+ feb-ui - 项目根目录
  + demo - 组件演示目录
  + dev  - 通用组件源码根目录
    + README.md - 说明文档
  + icon - ICON组件源码根目录

## 五、指令说明
### （一）安装整个项目所需的依赖
```shell
npm run install
```
### （二）生成路由文件
```shell
npm run route
```
### （三）开始调试（执行前会自动更新）
```shell
npm run dev
```
### （四）更新（=生成路由文件）
```shell
npm run update
```

### （五）创建代码片段
```shell
npm run snippets --author=<作者>
```
